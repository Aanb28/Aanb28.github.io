Learning Outcomes

Understand the applicability and challenges associated with different datasets for the use of machine learning algorithms.

Simple Perceptron

  1. Parameters:
    • Learning Rate: I find that a larger learning rate leads to overshooting the optimal solution but also converges more quickly. Although it takes longer to converge, a lower learning rate is more stable.
  2. Operator AND Perceptron
    • Modifications to Data:
      • Feature Representation: You are aware that binary features are assumed by the AND operator. The model must be modified to account for changes in feature representation or the inclusion of additional features.
    • Model Parameters:
      • Weights: The AND operator's weights are normally set to -2 for the bias and 1 for both inputs. The decision boundary changes when these values are changed.

Multi-Layer Perceptron (Sigmoid Function)

  • Modifications to Data:
    • Feature Scaling: I find that performance is enhanced by feature scaling.
    • Noise: Noise also has an impact on the model's accuracy. Techniques for noise reduction and data cleansing are crucial.
  • Model Parameters:
    • Learning Rate: Similar to the simple perceptron, convergence speed and stability are influenced by the learning rate.
    • Initialization: The convergence path and ultimate solution are impacted by the initialization of weights and biases. The convergence was impacted when I experimented with random initialization.
    • Hidden Layers: I found that the capacity and complexity of the model are influenced by the number of hidden layers and neurons in each layer. Although I noticed that the model is occasionally overfitting, extra layers and neurons may be able to learn more intricate patterns.
    • Activation Function: I also discovered that the model's capacity to learn non-linear relationships is impacted by the activation function I used (such as ReLU or tanh).

The performance of perceptron models is generally greatly impacted by modifications to the data or model parameters. Finding the best configuration for a particular problem frequently requires experimentation and parameter modification.
