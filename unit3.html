Changes in Correlation

  • Fortifying Correlation: when the variables are changed, the correlation coefficient rises and the relationship between them becomes more linear and consistent. This indicates that there is a stronger correlation between the variables.
  • Diminished Correlation: the correlation coefficient decreased as a result of the modifications I made to the data point that made the link between the variables less linear or constant. Sometimes it even gets close to zero, which means there is no association.

Changes in Regression

  • Regression Line Shifting: I noticed that as the variables changed, the regression line would sometimes move sideways, upward, or downward. This indicates a change in the dependent variable's expected values.
  • Modifying the Slope: This modifies the regression line's slope, which shows how quickly the dependent variable changes in relation to the independent variable. By randomly altering the data point, a steeper slope that denotes a stronger relationship and a flatter slope that suggests a weaker one have been seen.
  • Altering the Intercept: In certain situations, the regression line's intercept, which shows the dependent variable's expected value when the independent variable is zero, even shifts.

In summary, altering variables in data points have a profound effect on correlation and regression analysis.
